# LLM Research Platform

A comprehensive platform for collecting research data on user interactions with state-of-the-art Large Language Models (LLMs).

## Features

- **Multi-LLM Integration**: Support for OpenAI GPT, DeepSeek, Grok, and Anthropic Claude
- **Interaction Logging**: Automatic logging of all conversations to Google Sheets
- **Rating System**: User ratings and feedback collection
- **Analytics Dashboard**: Real-time analytics and data visualization
- **RESTful API**: Clean, documented API for data access

## Quick Start
